\subsubsection{Math}
\label{sec-Math}
\index{Math (package)}

{\bf Package}

\begin{verbatim}
import Math :: * ;
\end{verbatim}



{\bf Description}

Import this package to import the following packages instead of
importing the packages individually. 

\begin{verbatim}
import Complex::*;
import FixedPoint::*;
import NumberTypes::*;
import Divide::*;
import SquareRoot::*;
import FloatingPoint::*;
\end{verbatim}
